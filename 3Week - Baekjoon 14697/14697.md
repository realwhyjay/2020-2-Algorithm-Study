# 백준 14697 : 방배정하기

#### Date : 20/10/22

#### Lang : C++ 

#### Question : [백준 14697](https://www.acmicpc.net/problem/14697)

```c++
#include <iostream>
using namespace std;

int main() {
    int a,b,c,student;
    cin >> a >> b >> c >> student;

    for(int i=0; i<=50; i++){
        int studentA = a*i;
        
        for(int j=0; j<=50; j++){
            int studentB = b*j;

            for(int q=0; q<=50; q++){
                int studentC = c*q;

                if (studentA+studentB+studentC == student){
                    cout << 1 << endl;
                    return 0;
                }
            }
        }
    }
   cout << 0 << endl;
}
```
